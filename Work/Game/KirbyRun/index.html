Kirby Run






This is a 2D endless runner game made with Unity.
Controls:
- Up arrow to jump
- Eat apple to activate strong mode!
Gameplay:
Kirby will run faster and faster as Kirby runs longer. Avoid touching the enemies! However, eating the apples can activate strong mode temporarily. While in strong mode, Kirby is not afraid of any enemy!
