Centipede






This is a solo game clone project made from scratch of the arcade game Centipede using Game Maker Studio 2.
Controls:
- Mouse position to control the position of the player
- Click to shoot
Gameplay:
Shoot at the centipede! The centipede will go down by 1 row for each of the mushroom they encounter.
The part of the centipede being shot will turn into a mushroom as well.
Ps: Mushroom is shootable too!
